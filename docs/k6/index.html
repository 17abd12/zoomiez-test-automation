Commands

All load tests + reports
cd Zoomiez-test-Documentation
npm run test:k6
Individual scenarios
npm run test:k6:login
npm run test:k6:papers
npm run test:k6:dashboard
Site screenshots (k6 browser)
npm run test:k6:screenshots

# With auth (authenticated pages):
SCREENSHOT_EMAIL=you@example.com \
SCREENSHOT_PASSWORD=yourpass \
npm run test:k6:screenshots
With custom VUs / duration
K6_VUS=50 K6_DURATION=60s \
npm run test:k6:login
