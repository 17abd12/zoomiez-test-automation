CI Workflows

🔄 playwright.yml
Runs all Playwright E2E on push/PR to main. Starts MongoDB + backend + frontend as services. Uploads Allure results artifact.
Trigger: push / PR to main
🐍 backend-tests.yml
Runs pytest (40+ tests) then Newman contract tests in sequence. Uses test MongoDB. Mocks all paid APIs.
Trigger: push / PR to main
⚡ k6.yml
Nightly load tests at 2am UTC. Enforces p95 SLO thresholds. Also triggerable manually with custom VUs + duration.
Trigger: cron 0 2 * * * + manual
📊 allure-pages.yml
Downloads all Allure artifacts from test workflows, generates unified Allure report, publishes to GitHub Pages.
Trigger: after test workflows complete
✅ TypeScript Check
Runs tsc --noEmit on the Frontend project to catch type errors before any E2E tests run.
Part of allure-pages.yml

Auth Strategy

JWT Mint Trick (default)
Tests sign JWTs with JWT_SECRET_KEY directly using HS256. Token injected into Redux localStorage (app_state). No real login calls, no Resend OTP emails triggered. Auth setup runs once per role, saves storageState files to .auth/.
Real Login (REAL_LOGIN=true)
Set REAL_LOGIN=true in .env and fill real credentials in fixtures/users.json. Auth setup uses the actual login UI. Requires test accounts to exist in the database. Useful for smoke-testing auth flows end-to-end.
