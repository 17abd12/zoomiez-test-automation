Auth Strategy

JWT Mint Trick (default)
Tests sign JWTs with JWT_SECRET_KEY directly using HS256. Token injected into Redux localStorage (app_state key, with isLoggedIn: true). No real login calls, no Resend OTP emails triggered. Auth setup runs once per role, saves storageState files to .auth/.
Real Login (REAL_LOGIN=true)
Set REAL_LOGIN=true in .env and fill real credentials in fixtures/users.json. Auth setup uses the actual login UI. Requires test accounts to exist in the database. Useful for smoke-testing auth flows end-to-end.
